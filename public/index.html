Overview

Why

Human and machine – each on their own – won’t be enough to drive businesses in the coming decades. Tomorrow’s leading enterprises will be those that know how to meld the two effectively

What

Connect ambulances with hospitals, personal health data and traffic info to relay information while in route, to facilitate faster and more well-informed decisions

How

Use IBM Bluemix as a platform to develop an app, connect with wearables such as Google Glasses and integrate with Google APIs

Configure Google Account

Create Project & Enable APIs

  • Use your existing Google Account or create a new one
  • Upgrade to Google+ account
  • Go to Google API Manager & sign in
  • Click on the Create project button
  • Fill in the Project name field and click Create
  • Click on the Overview button in the left-side menu
  • Enable the following APIs(Click Enable API for each API):
    • Google Maps Geocoding API
    • Google Mirror API
    • Google+ API

Create Credentials

  • Click on the Credentials button in the left-side menu
  • Click Create credentials
  • Choose OAuth client ID
  • Click Configure consent screen button. If you must, set a product name on the consent screen
  • Fill in the Product name shown to users field with a name, i.e ConnectedAmbulanceApp
  • Click the Save button and it will redirect you back to where you left off.
  • Choose Application Type to be Web application
  • Name = ConnectedAmbulanceApp
  • In Authorized Javascript Origins, add the following addresses:
    • https://mirror-api-playground.appspot.com
    • http://yourAppName.mybluemix.net
  • In Authorized Redirect URIs, add the following address:
    • http://yourAppName.net/oauth2callback
  • Click Create, you will then get a pop-up with your client ID and client secret. Save these in a temporary file, so that they are easy to access. These credentials is needed in the next stage of this setup guide.

Provide Credentials

  • Copy & Paste the client ID & client secret in to the associated input-boxes on the right-side.
  • Click the Save ID & Secret button when done.

Setup Google Glasses (Optional)

Note: If the Google Glasses have not been used with the Google Account used in the above steps you most likely have to factory reset the glasses. After the glasses are powered on, go to Settings > Info about Glasses > Tap > Factory reset. Also make sure that you are logged in on the same account on the smartphone before pairing the newly factory reset glasses with it.

  • Make sure your phone has internet access and Bluetooth is enabled
  • Download and open the MyGlass app and login with the same Google account as used earlier
  • Pair the glasses with the phone. If your glasses is already paired with an account, you have to factory reset the glasses
  • Connect the glasses to Wi-Fi:
    • In the MyGlass app, click on the Wi-Fi symbol in the app header
    • Select a network or add a new one

Run the demo

Dispatch an Ambulance

  • Go to the Operations view.
  • Right click on a position in the map, this will create a marker
  • Left click on the marker
  • A pop-up will appear, similar to the one in the image
  • Use Street View to find the location where you want to dispatch the ambulance. When you are satisfied, click the Dispatch Ambulance button

Send Patient Information

  • Left click on the red cross in the map
  • A pop-up will appear, similar to the one in the image
  • Fill in the field Patient name with the value John Doe, or any other patients listed in View Patients
  • Click the Magnifier Icon button on the right side, it should display i.e. John Doe M/39
  • Fill in the field Short description of incident
  • Click the Send patient information button

Check if the application have sent cards to your Google account

  • Go to Google Mirror API Playground
  • Fill in your Client ID which you provided in the Provide Credentials step of Configure Google Account
  • Click the AUTHORIZE button
  • You should see a card with the text "Patient John Doe Critical medical data", similar to the first image on the right side
  • Click the Timeline tab, located under the card, in Google Mirror API Playground
  • Click on the card with an image and the text "Emergency response request", similar to the second image on the right side
  • The configuration have been successful if you have seen both cards

Customization (Optional)

Setting the default start location

Search or navigate to the location that you want to be default for the application. When you are satisfied right click on the location, a marker should then appear.
NOTE: If you use the Search Box, it will automatically create a marker for you. You can of course create a new marker.

Add Patient
